## Project Description ğŸ¤¯

**Finora** is an AI-powered personal finance and expense tracking app that automates financial management from end to end. Users can securely log in, create or edit transactions, and upload receipts which are scanned by AI to extract details like amount, date, and category. Finora visualizes data with interactive charts and aggregates insights using MongoDBâ€™s powerful aggregation pipelines. It also generates and emails monthly reports automatically.

---

## Project Value ğŸ’°

Finora is built for **individuals, freelancers, and small business owners** who want to save time managing their finances. By automating repetitive financial tasks like data entry, recurring payments, and reporting, Finora delivers clarity and control over cash flow. The app provides users with actionable insights, reduces errors, and helps them make better financial decisions without requiring accounting expertise.
